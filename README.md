# Road Club — Сайт для любителей велоспорта

Адаптивный лендинг о шоссейных, гравельных и ТТ велосипедах.

### Технологии:
* HTML5 (семантическая верстка)
* CSS3 (Flexbox, Grid, Custom Properties, логические свойства)
* Методология БЭМ

### Особенности:
* Плавный скролл к секциям.
* Доступность (A11y): работа с клавиатуры, ARIA-атрибуты.
* Чистая структура кода с соблюдением порядка свойств.
